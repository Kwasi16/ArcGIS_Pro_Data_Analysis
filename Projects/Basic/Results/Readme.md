# ArcGIS Pro Basic Projects Portfolio

This portfolio showcases **expertise in foundational GIS workflows and spatial data analysis** using ArcGIS Pro. The projects highlight proficiency in managing geospatial data, navigating maps, symbolizing layers, working with coordinate systems, querying features, labeling, and integrating tabular data into geospatial workflows.

## Key Skills Demonstrated

* **ArcGIS Pro Project Management**

  * Creating projects, managing folder connections, and structuring geodatabases
* **Map Navigation & Data Exploration**

  * Adding and navigating layers, viewing feature attributes, and selecting features interactively
* **GIS Data Models & Data Management**

  * Working with shapefiles, geodatabase feature classes, and ArcGIS Online data
  * Populating geodatabases, importing/exporting data, and converting tabular or online data
* **Coordinate Systems & Projection**

  * Differentiating coordinate system types
  * Performing geographic transformations and projecting data correctly
* **Layer Symbology & Visualization**

  * Symbolizing features using unique values, graduated colors, graduated symbols, and density-based methods
* **Feature Selection & Queries**

  * Performing attribute and spatial queries
  * Using definition queries and scale-based symbol classes for controlled feature visibility
* **Map Labels & Annotation**

  * Adding and modifying labels, creating label classes, and managing geodatabase annotation
* **Tabular Data Integration**

  * Preparing tabular data, creating joins and relates, and exporting joined layers
* **Spatial Data Handling & Analysis Preparation**

  * Managing raster and vector data to support downstream analysis

## Project Highlights

* Transform raw geospatial data into meaningful insights
* Apply GIS best practices for data organization, visualization, and attribute management
* Prepare data for advanced spatial analysis and mapping workflows
* Demonstrate practical proficiency with ArcGIS Pro interface and tools
